Descripción del Programa

Este programa es una interfaz web interactiva diseñada para calcular la probabilidad del resultado de un test médico (ya sea "Positivo" o "Negativo") en función de ciertas características del paciente. A continuación, se detallan sus componentes y funcionamiento:

1. Tabla de Datos de Pacientes

En la parte superior, se muestra una tabla con un conjunto de datos que representan registros de pacientes. Cada fila corresponde a un paciente y contiene la siguiente información:

  • Género: Masculino o Femenino.
  • Fumador: Sí o No.
  • Actividad Física: Nivel de actividad física del paciente (por ejemplo, Alta o Media).
  • Tiene Enfermedad: Indica si el paciente tiene alguna enfermedad (Sí o No).
  • Resultado Test: Resultado del test médico realizado al paciente (Positivo o Negativo).

Esta tabla sirve como base de datos de entrenamiento para inferir probabilidades con base en nuevos datos ingresados por el usuario.

2. Formulario de Predicción

Debajo de la tabla hay un formulario titulado: “Hallar la probabilidad del resultado dado”.

Este formulario permite al usuario seleccionar una combinación de características del paciente, incluyendo:

  • Género: Puede elegir entre "Masculino", "Femenino" o "Ninguno".
  • Fumador: Selección entre "Sí" o "No".
  • Actividad Física: Nivel de actividad física (por ejemplo, Alta, Media, etc.).
  • Tiene Enfermedad: Indica si se considera que el paciente tiene o no una enfermedad.

Una vez seleccionados estos valores, el usuario puede hacer clic en el botón “Filtrar”.

3. Cálculo de la Probabilidad

Al presionar el botón "Filtrar", el sistema calcula la probabilidad de que el resultado del test sea “Positivo” o “Negativo” para un paciente con las características seleccionadas.

Este cálculo probablemente se basa en el Teorema de Bayes, una técnica estadística que permite estimar probabilidades condicionales a partir de datos históricos. Es decir, estima: 

P(Resultado | Características del paciente)

Por ejemplo, si seleccionas:

  • Fumador: Sí
  • Actividad Física: Alta
  • Tiene Enfermedad: No
El sistema buscará en la tabla cuántos pacientes cumplen esas condiciones y cuántos de ellos obtuvieron un resultado positivo o negativo, y con eso calculará la probabilidad condicional de cada resultado.
